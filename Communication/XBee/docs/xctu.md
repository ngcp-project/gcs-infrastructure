# Testing XBee RF Modules with XCTU

This guide provides instructions to test the connection between two XBee RF modules. If the modules are able to communicaate through XCTU, they should also work with the XBee library.

## Requirements
1. 2x XBee modules with antennas
2. 1 - 2 laptops (2 XBees can be plugged into the same laptop)

## Instructions

1. Plug in XBee modules
